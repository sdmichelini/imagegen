About ImageGen Workflow

ImageGen organizes work using three core concepts.

Brand

A brand is reusable text guidance that defines style, voice, and constraints for generated images.

Stored as Markdown in ~/.imagegen/brands/*.md.

Project

A project is a container for related image work and can set a default brand context.

Stored under ~/.imagegen/projects/<slug>/project.json.

Work Item

A work item is a specific asset task inside a project, for example an icon for recipe-buddy.

Stored under work-items/<slug>/ with prompt and generated assets.
